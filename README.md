# Modern Productivity Dashboard 🚀

A sleek, responsive personal dashboard built with pure HTML, CSS, and JavaScript. This project demonstrates clean UI design using Glassmorphism and core JavaScript concepts.

## ✨ Features
- **Real-time Digital Clock**: Syncs with local system time.
- **Dynamic Weather Widget**: Simulated API integration for weather tracking.
- **Interactive To-Do List**: Add and remove daily tasks with smooth animations.
- **Responsive Design**: Fully functional on desktop, tablet, and mobile devices.

## 🛠️ Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Advanced styling with Flexbox, Glassmorphism, and Animations.
- **JavaScript (ES6+)**: DOM manipulation and real-time updates.
